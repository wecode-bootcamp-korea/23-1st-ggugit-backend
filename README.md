-  집밥을 특별하게! 요리키트 배달 서비스 [쿡킷](https://www.cjcookit.com) 클론 프로젝트

## 적용 기술 및 구현 기능

### 적용 기술

> - Back-End : Python, Django web framework, Bcrypt, My SQL, JWT
> - Common : AWS(EC2,RDS), RESTful API



### 구현 기능

- 회원가입 / 로그인
- 상품목록 조건별(맛별,나라별) 필터링, 신메뉴순-판매량순-높은가격순-낮은가격순 정렬
- 상품상세페이지 
- 상픔이름검색
- 장바구니 추가, 수정, 삭제 
- 메인페이지 배너 관리
